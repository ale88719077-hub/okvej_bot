OKVEJ BOT v15.0 — OPENAI AI-ПОМІЧНИК

Що додано:
- Кнопка «🤖 AI Помічник» у головному меню.
- Команда /ai.
- Підбір реальних товарів із каталогу Хорошоп, які є в наявності.
- Розуміння запитів звичайною мовою: смак, бюджет, подарунок тощо.
- Кнопки товарів і швидке додавання в кошик.
- Пояснення, як оформити замовлення через кошик.
- /cancel та кнопка завершення AI-діалогу.
- AI не повинен вигадувати товари, ціни або наявність.

Railway Variables:
OPENAI_API_KEY = ваш секретний ключ OpenAI
OPENAI_MODEL = gpt-5.6-luna   (необов'язково; це значення використовується за замовчуванням)

Важливо:
1. API OpenAI оплачується окремо від підписки ChatGPT.
2. На платформі OpenAI має бути налаштований Billing або доступний кредит.
3. Секретний ключ не додавайте в GitHub і не надсилайте в чат.
4. Після завантаження файлів зробіть Commit і Redeploy у Railway.

Перевірка після запуску:
1. Надішліть /version — має показати 15.0.
2. Натисніть «🤖 AI Помічник».
3. Напишіть: «Порадь цукерки з арахісом до 300 грн».
4. AI має відповісти і показати кнопки реальних товарів.
