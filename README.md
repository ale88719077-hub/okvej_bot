# OKVEJ Telegram Bot v12.0

- категории определяются по первому слову названия;
- товары `Цукерки ...` больше не попадают в `Шоколад`;
- новинки выбираются вручную администратором;
- под карточкой администратора есть `Додати/Прибрати з новинок`;
- команда `/admin` показывает Telegram ID и статус доступа;
- для доступа задайте Railway variable `ADMIN_USER_ID`;
- список хранится в `/data/manual_new_products.json`, если доступна папка `/data`;
- без Volume список может сброситься после Redeploy.
