# OKVEJ Telegram Bot v10.1

## Новое

- поиск понимает кириллицу и латиницу;
- `сник` находит `Snickers`;
- `рошен` находит `Roshen`;
- `komunarka` может находить `Комунарка`;
- поиск по артикулу сохранён;
- результаты сортируются по релевантности;
- каталог, корзина, избранное, просмотренные и `/пост` сохранены.
