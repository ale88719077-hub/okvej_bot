# OKVEJ Telegram Bot v13.0

## Чистая версия

- каталог, поиск, карточки, корзина, избранное и история просмотров сохранены;
- категории определяются по первому слову названия;
- ручные новинки и хиты управляются из карточки товара;
- админ определяется только через `ADMIN_USER_ID`;
- диагностическая команда `/admin_debug` удалена;
- данные хранятся в одном файле `admin_data.json`;
- предусмотрен раздел `recommended` на будущее;
- команда `/admin` показывает состояние админ-панели.

## Railway variables

- `ADMIN_USER_ID=210563345`
- `ADMIN_DATA_PATH=/data/admin_data.json`

Важно: без постоянного Railway Volume файл может сброситься после Redeploy.
