# OKVEJ Telegram Bot v13.2

## Метки в карточке товара

- товар из раздела `Хіти` показывает `🔥 ХІТ ПРОДАЖУ 🔥`;
- товар из раздела `Новинки` показывает `🆕 НОВИНКА`;
- товар из раздела `Рекомендовані` показывает `⭐ РЕКОМЕНДОВАНО`;
- метки появляются во всех карточках: каталог, поиск, избранное и история;
- ручное управление новинками и хитами сохранено.
