# OKVEJ Telegram Bot v13.4

## Каталог-карусель

- категория открывается сразу карточкой товара с большим фото;
- товары листаются кнопками `⬅️ Попередній` и `Наступний ➡️`;
- номер товара отображается как `1/152`;
- корзина, избранное, новинки, хиты и ссылка на сайт остаются в карточке;
- длинный текстовый список товаров больше не используется при открытии категории.
