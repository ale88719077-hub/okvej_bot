# OKVEJ Telegram Bot v5

## Новое в v5

- команда `/version` показывает установленную версию;
- кнопка `🔄 Оновити каталог` очищает кеш и заново получает товары;
- каталог из Horoshop API;
- показываются только товары с `presence.id == 1`;
- категории и количество товаров;
- пагинация по 8 товаров;
- поиск, корзина, оформление заказа и `/пост` сохранены;
- `/debug_stock` удалена;
- `.DS_Store` и `__pycache__` исключены из Git.

После деплоя проверьте `/version`: должна отображаться версия 5.0.
