# OKVEJ Telegram Bot v11.3

## Новинки по дате

- бот больше не использует значок `Новинка`;
- сначала проверяет `lastmod` в sitemap сайта;
- товары с датой за последние 30 дней попадают в новинки;
- если даты нет, бот запоминает дату первого появления товара;
- первый запуск создаёт базовую точку и не помечает весь каталог новинками;
- состояние сохраняется в `/data/okvej_new_products.json`;
- для сохранения между перезапусками Railway желательно подключить Volume к `/data`.
