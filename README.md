# OKVEJ Telegram Bot v11.4

## Исправление новинок

- бот снова использует реальные отметки `Новинка` на сайте;
- поиск метки выполняется рядом с артикулом и основной карточкой товара;
- блоки рекомендаций и похожих товаров отсекаются;
- товары DEMI и другие позиции с меткой на основной карточке должны определяться;
- проверяются только товары в наличии;
- результат кешируется на 6 часов;
- Volume и локальная база дат больше не нужны.
